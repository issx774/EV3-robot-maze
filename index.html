FSE 100 Learning Guide

🏗️ Engineering Design Process

Define → Research → Brainstorm → Build → Test → Iterate.

🤖 Maze-Solving Algorithms

Wall Follower: Hug one wall.
Pledge: Wall following + direction counter.
Dead-End Fill: Remember and skip dead ends.

📐 Key Concepts

Gear Ratios: Speed ↔ torque trade.
PID Control: Smooth turns.
Sensor Fusion: Ultrasonic + gyro + color.

💡 Pro Tips

• Calibrate sensors before every run
• Consistent wheel base width
• Account for battery voltage drop
• Test on actual maze surface
• Modular code: drive, turn, sense

🎯 Grading

• Maze completion time • Build quality • Code organization • Teamwork • Presentation
